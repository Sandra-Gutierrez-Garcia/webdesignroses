# Web Design Roses

## Espanol

### Sobre este proyecto
Este proyecto lo hice como una pagina principal para presentar mi trabajo de diseno web y ofrecer comisiones, especialmente pensado para usarlo en Tumblr. La idea es tener un espacio visualmente cuidado donde pueda mostrar mi estilo, explicar lo que hago y dar una forma sencilla de contacto a las personas que quieran encargarme una pagina.

Quise que la presentacion fuera simple, elegante y facil de entender. Por eso use una estetica suave en tonos violetas, una tipografia con personalidad y una estructura clara desde el primer vistazo.

### Mi idea para esta pagina
Quiero que esta pagina funcione como una especie de carta de presentacion. No solo sirve para que alguien vea mi estilo, sino tambien para que entienda que hago disenos personalizados, que acepto comisiones y que cada proyecto puede adaptarse a lo que necesita cada cliente.

Esta pagina esta pensada para:
- presentarme como disenadora
- mostrar mi estilo visual
- recibir comisiones o pedidos personalizados
- conectar mi trabajo de Tumblr con una pagina mas profesional

### Lo que incluye ahora
Actualmente la pagina tiene:
- un menu de navegacion con secciones como Home, About, Portfolio, Commissions y Contact
- una seccion principal con una descripcion breve de mi trabajo
- un boton para cambiar entre ingles y espanol
- HTML, CSS y JavaScript en un solo archivo
- una identidad visual limpia, delicada y centrada en la tipografia

### Para que quiero usarla en Tumblr
Mi idea es usar esta pagina como una pagina personalizada dentro de Tumblr o como enlace externo desde mi perfil. Asi, cuando alguien vea mi contenido y quiera pedirme una comision, puede entrar aqui y entender rapidamente que ofrezco, como es mi estilo y como contactarme.

Tambien me sirve para organizar mejor la informacion y separar mi trabajo de comisiones de otras publicaciones mas casuales.

### Lo que me gustaria agregar despues
Mas adelante me gustaria ampliar este proyecto con:
- una galeria o portafolio con ejemplos de trabajos
- una seccion de precios o paquetes
- un apartado con condiciones de trabajo
- una forma de contacto mas completa
- informacion mas detallada sobre el proceso de comision

### Estructura del proyecto
- index.html: pagina principal del proyecto, con la estructura, estilos y cambio de idioma

## English

### About this project
I made this project as a main page to present my web design work and offer commissions, especially with Tumblr in mind. The idea is to have a visually curated space where I can show my style, explain what I do, and give people a simple way to contact me if they want to commission a custom page.

I wanted the presentation to feel simple, elegant, and easy to understand. That is why I used a soft violet aesthetic, typography with personality, and a clear structure from the very first look.

### My goal for this page
I want this page to work as a kind of presentation card. It is not only meant to show my style, but also to explain that I create custom designs, that I take commissions, and that each project can be adapted to what each client needs.

This page is meant to:
- introduce me as a designer
- show my visual style
- receive commissions or custom requests
- connect my Tumblr work with a more professional-looking page

### What it currently includes
At the moment, the page includes:
- a navigation menu with sections such as Home, About, Portfolio, Commissions, and Contact
- a main hero section with a short description of my work
- a button to switch between English and Spanish
- HTML, CSS, and JavaScript in a single file
- a clean and delicate visual identity focused on typography

### How I want to use it on Tumblr
My idea is to use this page as a custom Tumblr page or as an external link from my profile. That way, when someone finds my content and wants to commission me, they can come here and quickly understand what I offer, what my style looks like, and how to contact me.

It also helps me organize my commission information better and separate it from more casual posts.

### What I would like to add later
Later on, I would like to expand this project with:
- a gallery or portfolio with work samples
- a pricing or packages section
- a terms of service section
- a more complete contact area
- more detailed information about the commission process

### Project structure
- index.html: main project page, including structure, styling, and language switch logic
